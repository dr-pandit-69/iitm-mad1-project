Steps to run the project:

1) Have python installed on your computer (if not see this video: https://www.youtube.com/watch?v=Kn1HF3oD19c)

2) Have pipenv installed on your computer, if not run this command in command prompt 'pip install pipenv'

3) Open the Code folder in File Explorer

4) type cmd in the search bar of opened Code folder 

5) type 'pipenv shell' in the opened command prompt, because it's always a good idea to run projects in virtual environments

6) Then run 'pip install requirements.txt' to install all project requirements

7) Now, finally type 'python3 app.py' to run the flask server, now go to your browser & type the URL http://127.0.0.1:5000 to access the website
